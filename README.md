### ☕ 스터디 진행 방식
- 매주 각자 한문제씩 선정(매주 금요일까지 문제 업로드)
- 총 5문제 풀이 업로드(Programmers에 정한 문제 폴더 업로드 해놓기)
- 문제 업로드 방식 (풀 문제안에 각자 폴더 만들어서 코드 업로드)
- 일주일에 1회(목요일 8시30분) 원격 회의로 코드 리뷰 진행
